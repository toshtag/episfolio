# ADR-0003: ライセンスに MIT を採用する

- **Status**: Accepted
- **Date**: 2026-04-28

## Context

OSS として公開するにあたり、ライセンスを選定する必要があった。採用障壁を下げ、個人・企業問わず利用しやすい形で公開することを優先する。

## Decision

**MIT License** を採用する。

## Alternatives Considered

- **Apache-2.0**: 特許条項が加わるが、MIT との実質的な差は小さく、ライセンス文書が長くなるだけ
- **AGPL-3.0**: ネットワーク経由での利用に対してもソース開示を求めるが、採用障壁が高くコミュニティの摩擦が増える
- **BUSL**: OSI 非承認であり、OSS として認識されないリスクがある
