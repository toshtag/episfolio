## 脆弱性の報告

セキュリティ上の問題を発見した場合、**GitHub Issues には投稿しないでください**。

以下のいずれかの方法でご連絡ください：

- GitHub の [Private vulnerability reporting](https://github.com/toshtag/episfolio/security/advisories/new)
- メール: mog2to4aki@gmail.com

報告内容には以下を含めてください：

- 問題の概要
- 再現手順
- 影響範囲（推定）
- 可能であれば PoC

報告受領後、7 日以内に初回返答を行います。
